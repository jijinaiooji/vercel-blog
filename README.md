# Vercel Blog

A simple blog built with Next.js and deployed on Vercel.

## Quick Start

1. Fork this repo or create a new Vercel project
2. Connect to Vercel
3. Deploy!

## Add New Posts

Edit `app/page.js` and add entries to the `posts` array.

## Tech Stack

- Next.js 14
- Vercel Hosting
- GitHub Actions for CI/CD
